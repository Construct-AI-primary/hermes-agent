# KnowledgeForge AI

Institutional memory and QA automation for multi-discipline engineering and construction expertise.

KnowledgeForge provides a layer of specialized agents that read from a shared knowledge base (~5k tagged documents across 50+ disciplines) to enable repeatable, automated code quality assurance and institutional learning.
