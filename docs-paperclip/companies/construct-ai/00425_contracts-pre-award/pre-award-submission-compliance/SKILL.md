# Pre-Award Submission Compliance

## Overview
Verification of tender submission completeness and compliance against all mandatory requirements before evaluation. Primary agent: 00425-004 Submission Compliance Agent.

## Steps
1. **Submission Receipt Verification** → Verify submission received before deadline, all required documents present
2. **Mandatory Compliance Check** → Check against mandatory requirements (bid security, signed declarations, valid certifications)
3. **Completeness Assessment** → Verify all sections completed, BOQ priced, technical submission present
4. **Compliance Report Generation** → Generate pass/fail compliance report for evaluation panel
5. **Clarification Management** → Request clarification for incomplete submissions if permitted

## Success Criteria
- All mandatory requirements verified
- Compliance report generated for evaluation panel
- Submissions classified as compliant or non-compliant
