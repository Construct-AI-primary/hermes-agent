# Commercial Tender Document Preparation

## Overview

Generate RFP/RFQ packages with evaluation criteria, scope injection, and commercial terms template population for competitive tendering processes.

**Announce at start:** "I'm using the commercial-tender-document-preparation skill to prepare tender documents."

## When to Use This Skill

**Trigger Conditions:**
- New procurement requirement requiring competitive tendering
- Re-tender for existing contract expiry
- Market test for new supplier identification
- Single-source justification requiring market analysis

**Prerequisites:**
- Scope of work/service defined
- Budget approved
- Evaluation criteria approved
- Shortlist of invited suppliers identified

## Step-by-Step Procedure

### Step 1: Select Tender Template

Choose appropriate tender template:
| Template | Use Case |
|----------|----------|
| RFQ (Request for Quotation) | Price-focused, standard specifications |
| RFP (Request for Proposal) | Solution-focused, complex requirements |
| EOI (Expression of Interest) | Market screening, pre-qualification |
| ITT (Invitation to Tender) | Formal competitive process |

### Step 2: Prepare Tender Documentation

1. Cover letter with project overview and submission instructions
2. Scope of work/service definition with measurable deliverables
3. Technical specifications and requirements
4. Commercial terms (payment, LDs, delivery, warranties)
5. Evaluation criteria and scoring methodology
6. Returnable schedule requirements
7. Contract form to be signed
8. Submission deadline and validity period

### Step 3: Populate Evaluation Criteria

Define and insert evaluation criteria:
- Price/Commercial (weight %)
- Technical capability (weight %)
- Past experience (weight %)
- Financial stability (weight %)
- Schedule/lead-time (weight %)
- HSSE compliance (weight %)

### Step 4: Generate Returnable Schedule

Define required supplier responses:
- Company profile and experience
- References from similar projects
- Technical approach and methodology
- Pricing schedule with breakdown
- Programme/schedule
- Resource allocation plan
- HSSE plan
- Financial statements

### Step 5: Issue Tender Package

1. Verify all documents complete and consistent
2. Distribute to shortlisted suppliers
3. Track receipt confirmations
4. Manage clarification queries
5. Issue addenda if necessary
6. Maintain tender logbook

## Success Criteria

- [ ] Appropriate tender template selected
- [ ] Scope of work/services complete and clear
- [ ] Commercial terms defined and inserted
- [ ] Evaluation criteria weighted and documented
- [ ] Returnable schedule defined
- [ ] Submission deadline and validity set
- [ ] Tender package distributed to all invited suppliers
- [ ] Clarification process managed
- [ ] Addenda issued if needed

## Common Pitfalls

1. **Inconsistent Scope vs Pricing** — Ensure the scope matches the pricing schedule format. Misalignment creates uncomparable quotes.
2. **Unweighted Criteria** — Without weights, evaluation becomes subjective. Always weight and publish criteria.
3. **No Validity Period** — Without a validity period, quotations can expire before award. Always include.
4. **Missing Return Schedule** — Suppliers may submit irrelevant information. Define exactly what must be returned.
