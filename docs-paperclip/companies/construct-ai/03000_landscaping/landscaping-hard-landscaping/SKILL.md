# Landscaping Hard Landscaping

## Overview
Documentation and specification for hard landscaping elements including paving, paths, walls, steps, fences, water features, lighting, and outdoor amenities. Primary agent: 03000-007 Hard Landscaping Agent.

## Steps
1. **Design Input Processing** → Receive hard landscaping drawings and specifications
2. **Material Identification** → Identify all hardscape materials with specifications
3. **Installation Specification Generation** → Generate installation specifications for each element type
4. **Quality Requirements** → Define quality standards and tolerances for each element
5. **Documentation Output** → Generate hardscape documentation package
