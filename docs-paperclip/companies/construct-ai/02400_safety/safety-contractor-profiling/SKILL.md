# Safety Contractor Profiling

## Overview
Assessment and monitoring of contractor safety performance using incident history, inspection results, compliance records, and safety management capability. Primary agent: Predictive Analytics Specialist.

## Steps
1. **Data Collection** → Gather incident history by contractor, inspection results, permit compliance, safety submissions
2. **Performance Analysis** → Calculate contractor safety score from multiple weighted factors
3. **Risk Classification** → Classify contractors: Excellent, Satisfactory, Needs Improvement, Unsatisfactory
4. **Profile Generation** → Create contractor safety profile with score trends and areas
5. **Recommendations** → Generate contractor-specific improvement plans or intervention
6. **Reporting** → Share contractor safety profile with procurement and management

## Success Criteria
- Performance accurately assessed with current data
- Risk classification drives appropriate intervention
- Profiles accessible to procurement and management

## Common Pitfalls
1. Insufficient data for new contractors → Use pre-qualification data
2. Not updating profiles regularly → Stale profiles don't reflect current performance
3. No intervention for underperforming → Profiles should drive action
