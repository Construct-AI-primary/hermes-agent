# Safety Dashboard Generation

## Overview
Generation of safety dashboards and reporting views from processed safety data. Includes key safety KPIs, incident trends, inspection status, CAPA tracking, PPE compliance, hazard trends, hotspot maps, and contractor safety scores. Primary agent: Predictive Analytics Specialist.

## Dashboard Components
- **Executive Overview** → Lagging and leading indicators summary
- **Incident Analysis** → Trends, severity, frequency, type distribution
- **Inspection Tracker** → Completion rate, findings by category
- **CAPA Status** → Overdue, closed, effectiveness verified
- **PPE Compliance** → Violations by type, location, trend
- **Hazard Monitoring** → Hazard detection trends, closure rates
- **Hotspot Analysis** → Incident hotspot maps and trends
- **Contractor Performance** → Safety scores and trends

## Success Criteria
- Data refreshed within required timeframes
- Dashboards accessible to appropriate users
- KPIs accurately reflect current safety status
- Drill-down capability available for all indicators
