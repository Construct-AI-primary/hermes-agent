# Safety Data Processing

## Overview
Processing, normalisation, and validation of safety data from multiple sources including incident reports, inspection results, sensor readings, PPE-detection events, and contractor safety records. Primary agent: Predictive Analytics Specialist.

## Data Sources Processed
- Incident reports (near-miss, LTI, medical treatment)
- Inspection findings and checklists
- Sensor readings (gas, noise, vibration, radiation)
- PPE-detection violations
- Hazard-detection events
- Contractor safety data
- Permit compliance records

## Success Criteria
- All data sources processed within required timeframes
- Data quality validated and anomalies flagged
- Schema consistent for analytics consumption
- Processing errors logged and alerting enabled
