# Weekly Review

Recurring weekly review of organisational health, progress, and priorities.

## Instructions

1. **Review Goals Progress**
   - Check status of organisation goals from COMPANY.md
   - Note completed, in-progress, and blocked items

2. **Team Health Check**
   - Review activity across all teams
   - Identify any agents or teams that appear blocked or idle
   - Check for any failures, blockers, or escalation items

3. **Knowledge Review**
   - Verify organisational knowledge is up to date
   - Check for any new standards or practices to add to `knowledge/`
   - Review any external discipline knowledge updates

4. **Task Backlog**
   - Review open tasks across the organisation
   - Prioritize upcoming work
   - Escalate any stale tasks that need attention

5. **Planning**
   - Identify priorities for the coming week
   - Delegate specific work to appropriate agents
   - Set clear expectations for deliverables
