# Trend Monitoring

## Overview
This skill defines how to systematically track and analyze trends across technology, markets, regulations, and industry practices. Proactive trend monitoring enables early response to changes and identifies opportunities before competitors.

## When to Use
- As an ongoing practice for strategic roles
- When entering a new domain or market
- When asked about emerging developments in a specific area
- When setting up regular monitoring for a team or company
- **Don't use when:** Monitoring only one specific, static topic with no expected change

## Core Procedures

### Step 1: Define Monitoring Scope
Select areas to monitor:
- **Technology:** New tools, frameworks, platforms, infrastructure
- **Market:** Customer needs, competitor moves, pricing, M&A
- **Regulation:** New laws, compliance requirements, standards
- **Industry:** Best practices, benchmarks, case studies, research
- **Social:** Workforce trends, cultural shifts, customer behavior
- **Economic:** Macro indicators, funding patterns, cost trends

### Step 2: Identify Sources
For each area:
- **Primary Sources:** Academic journals, government publications, company filings, standards bodies
- **Secondary Sources:** Industry reports, analyst firms, trade publications, podcasts
- **Community Sources:** Forums, GitHub trending, social media thought leaders, conferences
- **Internal Sources:** Customer feedback, agent observations, incident data

### Step 3: Set Up Monitoring
- Subscribe to key newsletters, RSS feeds, and alerts
- Set up Google Alerts or equivalent for key terms
- Follow relevant thought leaders and organizations
- Schedule regular reviews (weekly digest, monthly analysis)
- Track signals in structured format

### Step 4: Analyze Trends
For each signal detected:
```
TREND SIGNAL
============
Trend: [what's changing]
Category: [technology/market/regulation/industry/social/economic]
Source: [where observed]
Signal Strength: [weak/anecdotal/multiple-confirmed/mainstream]
Momentum: [emerging/growing/plateauing/declining]
Impact on Us: [none/low/medium/high/critical]
Time Horizon: [immediate/6-12months/1-3years/3+years]
```

### Step 5: Report and Act
- Monthly trend digest: Summary of signals and significance
- Quarterly trend analysis: Deeper assessment of key trends
- Annual trend outlook: Strategic implications and recommendations
- Immediate alert for critical trends that need rapid response

## Quality Checklist
- [ ] Monitoring scope clearly defined
- [ ] Multiple source types used for diversity
- [ ] Regular monitoring schedule established
- [ ] Signals documented with consistent format
- [ ] Impact assessment for each trend
- [ ] Reports delivered to decision-makers

## Error Handling
- **Error:** Monitoring becomes overwhelming with too many signals
  **Response:** Prune low-value sources, focus on high-impact areas, automate collection
- **Error:** Trend turns out to be hype not substance
  **Response:** Document the difference, improve signal filtering, share as lesson-learned
