# Competitive Analysis

## Overview
This skill defines how to systematically analyze competitors to understand their strengths, weaknesses, strategies, and market position. Competitive intelligence enables informed strategic decisions and identifies opportunities for differentiation.

## When to Use
- Before entering a new market or launching a new product
- When asked to assess competitive threats
- When evaluating market positioning or pricing
- When strategic planning or annual reviews
- When competitor announces significant change
- **Don't use when:** Market is new with no established competitors

## Core Procedures

### Step 1: Identify Competitors
Categorize competitors:
- **Direct:** Same product/service, same target market
- **Indirect:** Different approach to same problem
- **Potential:** Could enter your market easily
- **Substitute:** Customers might choose instead of your solution

### Step 2: Gather Intelligence
Collect on each competitor:
- **Product:** Features, quality, pricing, target segment
- **Market:** Market share, growth rate, geographic reach
- **Strategy:** Recent moves, partnerships, acquisitions, public statements
- **Strengths:** What they do better than anyone
- **Weaknesses:** Where they underperform or complain
- **Customer Perception:** Reviews, ratings, reputation, complaints

### Step 3: Analyze Positioning
Create competitive matrix:
| Dimension | Us | Competitor A | Competitor B |
|-----------|----:|--------------|--------------|
| [Key differentiator 1] | | | |
| [Key differentiator 2] | | | |
| [Key differentiator 3] | | | |
| Price point | | | |
| Market share | | | |
| Growth trajectory | | | |

### Step 4: Identify Opportunities
- **Gaps:** Underserved customer needs competitors aren't addressing
- **Weaknesses:** Competitor pain points we can exploit
- **Differentiation:** Unique value we offer that nobody else does
- **Threats:** Competitor advantages we need to counter
- **Trends:** Market shifts that favor or disadvantage us

### Step 5: Document and Share
```
COMPETITIVE INTELLIGENCE
========================
Competitor: [name]
Analysis Date: [date]
Key Findings: [3-5 bullet points]
Our Advantage: [where we win]
Their Advantage: [where they win]
Threat Level: [Low/Medium/High]
Recommended Action: [what we should do about this]
```

## Quality Checklist
- [ ] All relevant competitor types considered (direct, indirect, potential)
- [ ] Intelligence from verified sources (source-verification applied)
- [ ] Objective assessment (not biased toward our strengths)
- [ ] Specific actionable recommendations provided
- [ ] Intelligence is current (dated within last 6 months)

## Error Handling
- **Error:** Intelligence based on unverified rumors
  **Response:** Flag as unverified, seek primary sources, downgrade confidence level
- **Error:** Analysis becomes outdated quickly in fast-moving market
  **Response:** Set quarterly review schedule, trigger ad-hoc reviews when competitor makes major move
