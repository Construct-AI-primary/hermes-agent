# Landscaping Soil Analysis

## Overview
Processing and interpretation of soil test results to generate soil improvement recommendations for landscape areas. Primary agent: 03000-003 Soil Analysis Agent.

## Steps
1. **Test Data Input** → Receive laboratory soil test results (pH, nutrients, texture, organic matter)
2. **Interpretation** → Compare results against target ranges for plant types
3. **Recommendation Generation** → Generate specific recommendations for pH adjustment, nutrient supplementation, and organic matter improvement
4. **Application Rate Calculation** → Calculate application rates from area and current soil conditions
5. **Reporting** → Generate soil test report with improvement recommendations
