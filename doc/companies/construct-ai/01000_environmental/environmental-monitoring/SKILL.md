# Environmental Monitoring Skill

## Overview

This skill provides capability for environmental management including monitoring data collection, pollution control, waste management, biodiversity protection, regulatory compliance, and incident management. It supports Environmental Management (01000) in their operational responsibilities.

## Triggers

- Environmental monitoring data collection required
- Pollution incident response needed
- Waste management tracking required
- Regulatory compliance check needed
- Environmental report generation needed

## Prerequisites

- Access to environmental monitoring data
- Access to IoT sensor feeds
- Access to waste tracking system
- Access to incident management system
- Environmental Management domain knowledge loaded

## Steps

### Step 1: Collect Monitoring Data
- Gather air quality data (PM10, PM2.5, dust)
- Gather water quality data (pH, turbidity, hydrocarbons)
- Gather noise monitoring data
- Gather soil monitoring data
- Verify sensor calibration and data quality

### Step 2: Assess Environmental Compliance
- Compare monitoring data against regulatory limits
- Identify permit conditions status
- Check EIA/ESIA condition compliance
- Flag exceedances for investigation

### Step 3: Track Waste Performance
- Monitor waste generation rates
- Track waste segregation compliance
- Calculate recycling rate
- Verify disposal via licensed contractors

### Step 4: Manage Environmental Incidents
- Log and classify incident
- Initiate immediate response (containment)
- Guide investigation with root cause analysis
- Track corrective actions to closure
- Determine regulatory notification requirements

### Step 5: Generate Environmental Report
- Compile monitoring data summary
- Document compliance status
- Include incident summary and lessons learned
- Recommend improvement actions

## Success Criteria

- All environmental monitoring parameters collected
- Exceedances detected and flagged within 1 hour
- Waste tracking complete with reconciliation
- Incidents classified correctly with regulatory notification
- Report complete with all required sections

## Pitfalls

- Do not fabricate or modify monitoring data
- Do not override stop-work orders for environmental non-compliance
- Do not classify incidents at severity lower than evidence indicates
- Do not bypass regulatory notification requirements
- Do not present uncalibrated sensor data as valid
